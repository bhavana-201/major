# 🚀 Quick Start Guide

## Get Started in 3 Steps!

### Step 1: Install Flask
```bash
pip install flask
```

### Step 2: Run the Application
```bash
cd finance-agent
python app.py
```

### Step 3: Open in Browser
Navigate to: **http://localhost:5000**

---

## 🎯 First Time Usage

1. **Set Your Budget**
   - Default is ₹10,000
   - Change it in the left panel
   - Click "Update Budget"

2. **Add Your First Expense**
   - Enter amount: e.g., 250
   - Description: e.g., "Lunch at restaurant"
   - Pick today's date
   - Click "Add Expense"

3. **See AI in Action**
   - Watch automatic categorization
   - Check AI insights panel
   - View spending summary

4. **Chat with Your Agent**
   - Type: "How much have I spent?"
   - Type: "Give me a tip"
   - Type: "Show spending by category"
